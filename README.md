# First
Ты не думай,
            щурясь просто
из-под выпрямленных дуг.
Иди сюда,
         иди на перекресток
моих больших
            и неуклюжих рук.
Не хочешь?
          Оставайся и зимуй,
и это
     оскорбление
               на общий счет нанижем.
Я все равно
           тебя
               когда-нибудь возьму -
одну
    или вдвоем с Парижем.
